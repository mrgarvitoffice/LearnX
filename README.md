## 📘 About The Project

*LearnX* is a futuristic, AI-powered platform designed to revolutionize how learners of all ages study, revise, and grow. It automates material creation using *Google Gemini AI*, delivering seamless contextual learning and boosting retention — all through a clean, fast, and responsive UI powered by **Next.js**.

> Built for the next generation of global talent. LearnX is more than a project — it's a learning revolution.

---

## 🔑 Core Features

### 🤖 AI Content Forge
- AI-powered markdown notes with diagrams and visual prompts
- Instant MCQs nd Short Answer with negative marking and reasonin
- Dynamic neural flashcards from any topic

### 🧪 Custom Logic Lab
- Build custom assessments using topic selection, difficulty, and per-question timers
- Adaptive feedback loops based on performance

### 💬 Neural Link (Chatbot)
- Powered by *Google Gemini 2.5 Flash*
- Multiple expert personas (J.A.R.V.I.S., Coder, Helper)
- Full multimodality: Analyze PDF, Image, Video, or Audio context

### 🎧 Audio Factory
- Text-to-speech in natural voices for auditory revision
- Dual-voice dialogue generation for complex topics

### 📰 Intel Stream (Current Affairs)
- Real-time global headlines localized for your profile
- Categorized feeds for technical and academic focus

### 🛠 Data Vault & Toolkit
- 📐 Precision unit converter
- 🧮 Advanced scientific calculator
- 🔍 YouTube & Google Books semantic
